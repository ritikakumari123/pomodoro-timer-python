# Custom Pomodoro Timer

A simple and customizable Pomodoro Timer built using Python.

This project helps users stay focused and productive using the Pomodoro Technique — a time management method that divides work into study sessions and short breaks.

---

## ✨ Features

- Custom work duration
- Custom break duration
- Multiple study rounds
- Live countdown timer
- Session completion alerts
- Simple and beginner-friendly interface

---

## Technologies Used

- Python
- Time Module

---

## How It Works

1. Enter work time in minutes
2. Enter break time in minutes
3. Enter number of rounds
4. The timer starts counting down
5. Break sessions begin automatically after study sessions

---

## ▶️ How to Run

1. Install Python
2. Download the project files
3. Run the Python file

```bash
python pomodoro_timer.py
```
